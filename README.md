# 🧪 Proyecto de Práctica Git

Bienvenida, Raquel. Este proyecto está diseñado para practicar los comandos más importantes de Git paso a paso.

## 📁 Estructura del proyecto

```
git-practica/
├── README.md               ← estás aquí
├── src/
│   ├── app.js              ← archivo principal (para practicar cambios)
│   └── utils.js            ← utilidades (para practicar merge/rebase)
├── docs/
│   └── notas.md            ← documentación (para practicar ramas)
└── ejercicios/
    ├── 01-commits.md       ← Ejercicio 1: commits básicos
    ├── 02-ramas.md         ← Ejercicio 2: crear y gestionar ramas
    ├── 03-merge.md         ← Ejercicio 3: merge
    ├── 04-rebase.md        ← Ejercicio 4: rebase
    ├── 05-pull-push.md     ← Ejercicio 5: pull y push
    └── 06-pull-request.md  ← Ejercicio 6: pull requests y revisión
```

## 🚀 Cómo empezar

1. Crea un repositorio nuevo en GitHub llamado `git-practica`
2. Clónalo en tu ordenador con GitHub Desktop o con:
   ```bash
   git clone https://github.com/TU_USUARIO/git-practica.git
   ```
3. Copia todos estos archivos dentro de la carpeta clonada
4. Sigue los ejercicios en orden

## 📌 Comandos de referencia rápida

| Acción | Comando |
|--------|---------|
| Ver estado | `git status` |
| Ver historial | `git log --oneline` |
| Ver ramas | `git branch` |
| Cambiar de rama | `git checkout nombre-rama` |
| Crear y cambiar de rama | `git checkout -b nombre-rama` |
| Subir cambios | `git push origin nombre-rama` |
| Bajar cambios | `git pull origin main` |
